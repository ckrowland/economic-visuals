Economic Visuals

Create basic consumers and producers, adjust variables on the fly, and watch how an economy adjusts to the changes. Currently doesn't work on Safari or Firefox.

Basic Slider

A basic economy of consumers and producers. Consumers move to producers, get resources, travel home and consume the resources. Red means consumer is empty and looking for resources. Green means consumer has enough resources to consume. To remove a line from the statistics graph, click it's title in the legend.

Basic Editor

The same as basic slider, but you can place consumers and producers individually.

Basic Wave

This same as basic slider, but you can set parameters to follow a repeating wave.
